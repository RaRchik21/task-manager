# Task Tracker

Корпоративная система управления задачами с иерархией ролей и гибкими настройками доски.

---

## Основные возможности

### Канбан-доска
- Колонки создаются и настраиваются вручную — можно задать название и цвет каждой колонки
- Задачи перетаскиваются между колонками drag & drop
- Колонки также можно менять местами перетаскиванием
- Поиск задач по названию и комментарию в реальном времени

### Задачи
- Приоритет задачи — низкий 🟢 / средний 🟡 / высокий 🔴 (отображается цветной полоской на карточке)
- Дедлайн с цветовой индикацией — зелёный (далеко), жёлтый (скоро), красный (просрочено)
- Комментарий к задаче
- Метка **"В целях у Главного Специалиста"** — такие задачи выделяются красной рамкой и бейджем ⭐, они имеют наивысший визуальный приоритет
- Архивирование задач — задачи не удаляются, а уходят в архив, откуда их можно восстановить или удалить навсегда

### Фильтрация задач
- **Мои задачи** — только свои
- **Все задачи** — задачи всех подчинённых
- **По пользователю** — выбрать конкретного подчинённого из списка
> Фильтры доступны начиная с роли Специалист

### Дашборд 📊
- Общее количество задач, просроченных, в целях ГС, в архиве
- График активности за последние 7 дней
- Распределение задач по колонкам
- Распределение по приоритетам
- Топ сотрудников по количеству задач (для ГС и ВС)
> Дашборд доступен начиная с роли Ведущий Специалист

### Настройка внешнего вида
- 9 готовых тем с градиентными фонами (Песок, Лаванда, Мята, Персик, Сталь, Ночь и др.)
- Конструктор градиента — выбрать два цвета и угол
- Регулировка прозрачности шапки, колонок и карточек
- Индивидуальный цвет для шапки и карточек
- **Настройки применяются сразу у всех пользователей системы**
- Смена названия системы
> Настройки доступны начиная с роли Ведущий Специалист

## Видимость задач по ролям

| Роль | Чьи задачи видит |
|---|---|
| Младший Специалист | Только свои |
| Старший Специалист | Только свои |
| Специалист | Свои + задачи своих Младших Специалистов |
| Ведущий Специалист | Свои + задачи своих Специалистов и их Младших Специалистов |
| Главный Специалист | Все задачи всех сотрудников |

## Страница "Структура" 🏢

Визуальная органиграмма организации в виде дерева.

**Просмотр** доступен начиная с роли **Специалист**.

**Редактирование** доступно начиная с роли **Ведущий Специалист**:
- Назначить или изменить руководителя любого сотрудника
- Нельзя назначить руководителем того, кто стоит ниже по иерархии
- Главного Специалиста можно скрыть в раздел "Без руководителя"

> Структура отображает реальную иерархию — именно она определяет какие задачи видит каждый пользователь в фильтрах.

## Мелкие фишки

- 🎨 Настройки темы применяются **сразу у всех** — не нужно каждому выбирать вручную
- ⭐ Задачи "В целях ГС" всегда выделяются визуально и видны поверх обычных
- 📦 Задачи **не удаляются** — они уходят в архив. Удалить навсегда можно только вручную из архива
- 🔍 Поиск работает по названию и комментарию задачи одновременно
- 👥 В списке пользователей показываются только те, кто ниже тебя по роли — вышестоящих не видно в фильтрах
- 🏷️ Карточки чужих задач у Специалиста выделяются фиолетовой полоской — сразу видно что это не твоя задача
